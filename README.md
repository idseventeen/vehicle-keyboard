## vehicle-keyboard

车牌号码输入键盘，支持新能源车牌号码

### Usage

- use in VueComponent

```js
// a.vue
import VehicleKeyboard from 'vehicle-keyboard'
export default {
  component: {
    VehicleKeyboard
  }
}
```
- register Component
```js
// main.js
import VehicleKeyboard from 'vehicle-keyboard'
Vue.component(VehicleKeyboard.name, VehicleKeyboard)
```
